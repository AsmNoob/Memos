Introduction to Django
======================

Start a project
---------------

		django-admin.py startproject mysite

**NOTE**: on a server you should do it in the /var/www/ directory.

What is created:

		mysite/
		    manage.py
		    mysite/
		        __init__.py
		        settings.py
		        urls.py
		        wsgi.py


Setting up your database
------------------------

*The default database manager is Sqlite(included in python)*
